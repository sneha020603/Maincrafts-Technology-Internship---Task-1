# 🌐 Full Stack Contact Form | Sneha

🚀 This project is built as part of the **Maincrafts Technology Internship - Task 1**

---

## 📌 Project Overview

A modern **full-stack portfolio landing page** with a working contact form:

👉 Frontend collects user data  
👉 Backend processes and logs it  

This project demonstrates real-world **frontend + backend integration**

---

## 🛠️ Tech Stack

### 💻 Frontend:
- HTML5
- CSS3 (Modern UI + Animations)

### ⚙️ Backend:
- Java
- Spring Boot

---

## ✨ Features

✔ Responsive landing page  
✔ Modern UI with animations  
✔ Contact form (Name, Email, Message)  
✔ Spring Boot backend integration  
✔ POST request handling  
✔ Console logging of submitted data  
✔ Clean project structure
