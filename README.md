## ⚡ How to Run the Project

1️⃣ Clone the repository

```
git clone https://github.com/your-username/contactform.git
```

2️⃣ Navigate to project

```
cd contactform
```

3️⃣ Run Spring Boot

```

```

4️⃣ Open browser

```
http://localhost:8080
```
